About PySandbox WASM

A client-side code execution engine powered by WebAssembly. Scripts execute inside an isolated browser heap without sending sensitive code to remote cloud servers.

Execution Limits

  • • Max Memory Allocation: Browser JS Heap
  • • Network Access: Restricted Client-Side
  • • Python Version: 3.11 Runtime (WASM)

Troubleshooting

If you see Engine Failed, ensure you are running this project using a local server (e.g., Live Server in VS Code) rather than opening `index.html` as a raw local file (`file://`).
